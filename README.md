## person-opacity-image
实时获取视频中人物的透明base64图片

## Install
```bash
npm install @simon_he/person-opacity-image
```

## Usage
```js
import PersonOpacityImage from '@simon_he/person-opacity-image'
onMounted(() => getPersonOpacityImage(video.value, (base64) => {
  console.log(base64)
}))
```
